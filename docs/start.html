NEM Cold Wallet is based on official NEM Wallet project and focuses on providing increased security via the use of fully functional QR-based cold wallet.

Create new wallet

When you want to create new wallet simply click "sign-up" in the top-right corner.

Simple wallet

This type is fully functional wallet which allows you to use all features of the application. Wizzard will guide through the process and private key will be generated.

Security warning

Private key is for this wallet type stored on your computer encrypted by the password you provided. Use only on trusted and well-protected computers. Compromised computer will likely leak enough information that attacker will be able to make transactions from your account.

Private key wallet

If you want to create wallet for private key which you already have then use this wallet type. Fully functional wallet will be created using this method.

Security warning

Private key is for this wallet type stored on your computer encrypted by the password you provided. Use only on trusted and well-protected computers. Compromised computer will likely leak enough information that attacker will be able to make transactions from your account.

View only wallet

This wallet is created only by providing your NEM address. This is perfectly safe to be used on untrusted computers because NEM address is public information and can not be used to transfer funds.

Use Cold Wallet

Cold wallet is wallet which never comes online. As such it is perfectly safe because it can never leak your private key to the attacker. It can be used with offline simple or private key wallet and online view only wallet.

Never expose your private key

The only way to be 100% sure that your private key is safe is to never go online with it. The computer where you generate and store your private key can never get online.

Using the cold wallet

To use cold wallet you need 2 devices (computer, phone, tables) with camera wunning NEM Cold Wallet application. One device needs to be online and have View only wallet created. The other needs to be always offline with simple or private key wallet created. We will call this offline wallet Cold wallet.

Preparing the transaction

Use view only wallet online to prepare the transaction that you want to make. This can be any transaction with mosaic or for multisig account. When you press "Send" button you will be asked to provide access to your camera and dialog box will be displayed with 2 parts:

  • QR code representing the transaction you want to execute
  • Camera scanner for the response from the cold wallet
Leave the dialog box open, you will get back to it using the scanner from the Cold wallet.

Signing the transaction

Use the Cold wallet (on offline device) to sign the transaction as follows:

  1. Start the application but do not log-in
  2. Open the "Sign QR transaction" module in the page header section
  3. Select account you want to use to sign the transaction and enter password
  4. Click the "Import the QR of any transaction to sign" buton
  5. You will be asked to allow access to the camera for the scanner
  6. Scan the QR code from the online device (created in previous section by view wallet)
  7. After code is scanned, camera scanner disappears and new QR code is displayed by the Cold wallet

Sending the transaction

The QR code generated by the Cold wallet needs to be scanned back in the online View only wallet to submit the transaction.
